Testando Git/GitHub

Comandos Básicos GIT

git init (Cria um repositório git vazio ou reinicia um existente)
git clone (Copia um repositório)
git add ou git add --nome_arquivo-- (Adicionar conteúdo do arquivo ao índice)
git commit -m "mensagem de commit" (Registrar mudanças no repositório)
git push (Atualiza remotamente as referencias associando aos objetos)
git pull (Busca e integra com outro repositório local ou remoto)
git merge (Junta dois ou mais históricos de desenvolvimento)
git status (Mostra o trabalho da árvore de status)
git log (Mostrar logs de confirmação)
git branch --nome_branch-- (Listar, criar ou excluir ramificações)
fonte: https://git-scm.com/docs

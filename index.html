Configuração

Geral

As configurações do GIT são armazenadas no arquivo .gitconfig localizado dentro do diretório do usuário do Sistema Operacinal (Ex.: Windows: C:\Users\Documents and Settings\Beta ou *nix /home/beta).

As configurações realizadas através dos comandos abaixo serão incluídas no arquivo citado acima.

Setar usuário

git config --global user.name "Roberta Gaspar"

Setar email

git config --global user.email roberta.gaspar@fatec.sp.gov.br

Setar editor

git config --global core.editor vim

Setar ferramenta de merge

git config --global merge.tool vimdiff

Setar aquivos a serem iguinorados

git config --global core.excludesfile ~/.gitignore

Listar configurações

git config --list
